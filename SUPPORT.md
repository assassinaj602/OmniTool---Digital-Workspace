Need help with OmniTool? We're here for you!

## How to ask for help

### 1. Check the Documentation
Before asking, please check the [README](README.md) and tool-specific tips inside the application.

### 2. GitHub Discussions (Recommended)
For general questions, "how-to" guidance, or feature ideas, please use [GitHub Discussions](https://github.com/yourusername/omnitool/discussions). 
*   It keeps the Issues tracker clean for bugs.
*   The community can help answer questions.

### 3. GitHub Issues
If you believe you have found a **bug** or a **technical error**, please open an [Issue](https://github.com/yourusername/omnitool/issues).

## What to include in your question

To help us help you, please include:
*   Which tool you are using (e.g., "Image Resizer").
*   What browser and OS you are on (e.g., Chrome on Windows 11).
*   Screenshots or error messages (if applicable).
*   **Note:** Do not share private files. Create dummy files if you need to demonstrate an issue.

## Community Expectations

OmniTool is maintained by volunteers. 
*   Please be patient.
*   Be respectful (see [Code of Conduct](CODE_OF_CONDUCT.md)).
*   Give back to the community if you can!
